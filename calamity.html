Public Health and Economic Consequences of Natural Disasters

Synopsis

In this report we aim to describe the public health and economic problems in the USA, caused by storms and other severe weather events, using data from the year 1950 to 2011 . Regarding Public health, ‘Tornado’ is the most harmful event, followed by ‘Excessive Heat’. Tornados are responsible for 65% of all injuries and 37.19% of all fatalities. And Regarding Economic damage, ‘Flood’ is the most economically damaging event. Floods alone have caused 31.5% of the total economic damage, and it is followed by ‘Hurricane/Typhoon’ and ‘Tornado’.

Processing the data, to create a final dataset

This data needs to be processed in two ways:

  • As seen above, the data containes 37 columns in total. For the purpose of this report, we are only interested in 8 of them. Hence we need to take a subset of the data, containing the following columns:

    1. “STATE”: The State that the event took place in.
    2. “EVTYPE”: Type of Event.
    3. “FATALITIES”: Number of Fatalities from the event
    4. “INJURIES”: Number of Injuries from the event
    5. “PROPDMG”: Property damage in USD
    6. “PROPDMGEXP”: Magnitude of property damage
    7. “CROPDMG”: Crop damage in USD
    8. “CROPDMGEXP”: Magnitude of crop damage

  • There are two pairs of columns pertaining to the damage costs(“PROPDMG”, “PROPDMGEXP”, “CROPDMG”, “CROPDMGEXP”). For our final dataset, we need to combine each pair, into one single column, by multiplying the numeric value of the damage(eg. “PROPDMG”), to its magnitude/exponent(eg. “PROPDMGEXP” ).

We will begin by first creating the individual columns for each type of damage:

Before we multiply the PROPDMG and CROPDMG columns with the PROPDMGEXP and CROPDMGEXP columns respectively, we need to mutate the values of the *EXP columns, by replacing the characters and numbers in them, with their actual numeric values. Example: “B” = 10^9, “K” = 10^3, “5” = 10^5.

prop_exp <- as.vector(data$PROPDMGEXP);
crop_exp <- as.vector(data$CROPDMGEXP);

# Loop through the vector, and mutate its values.
clean <- function(v) {
    
    for(i in 1:length(v)) {
        if (v[i] == "1") {
            v[i] <- "10";
        } else if (v[i] == "2" || v[i] == "H" || v[i] == "h") {
            v[i] <- "100";
        } else if (v[i] == "3" || v[i] == "K" || v[i] == "k") {
            v[i] <- "1000";
        } else if (v[i] == "4") {
            v[i] <- "10000";
        } else if (v[i] == "5") {
            v[i] <- "100000";
        } else if (v[i] == "6" || v[i] == "M" || v[i] == "m") {
            v[i] <- "1000000";
        } else if (v[i] == "7") {
            v[i] <- "10000000";
        } else if (v[i] == "8") {
            v[i] <- "100000000";
        } else if (v[i] == "B") {
            v[i] <- "1000000000";
        } else if (v[i] == "0"){
            v[i] <- "1";
        } else {
            v[i] <- NA;
        }
    }
    v
}

prop_exp <- as.integer(clean(prop_exp));
crop_exp <- as.integer(clean(crop_exp));

Now that we have mutated PROPDMGEXP and CROPDMGEXP columns, we can multiply them with PROPDMG and CROPDMG respectively, and get two columns: property damage and crop damage, the values of each one of them being the actual damage done in USD(eg. 25000). We can then add these two columns, to get a single column for the total damage done.

PROPDMG <- data$PROPDMG * prop_exp;
CROPDMG <- data$CROPDMG * crop_exp;

prop_crop_dmg <- as.data.frame(cbind(PROPDMG, CROPDMG));

# Create a final data frame with PROPDMG, CROPDMG and total DMG. The total DMG needs 
# to have the value of CROPDMG + PROPDMG, and needed to be manipulated in case
# CROPDMG and/or PROPDMG have NA's.

total_dmg <- mutate(prop_crop_dmg, DMG = ifelse(is.na(PROPDMG) & is.na(CROPDMG), NA,
                       (ifelse(is.na(PROPDMG), 0, PROPDMG)) +
                        (ifelse(is.na(CROPDMG), 0, CROPDMG))
));

Answer

It is clear from the tables and plots above, that Tornado is the most harmful event with respect to population health. Since 1950 to 2011, it is responsible for 91346 injuries(65% of the total injuries), and 5633 fatalities(37.19% of the total fatalites), placing it at the No.1 position of both tables. Excessive Heat is also a dangerous event, it has caused a total of 1903 fatalities and 6525 injuries, making it No.2 on the fatalities table and No.4 on the injuries table.
